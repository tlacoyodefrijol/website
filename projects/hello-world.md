This is the first project entry. Edit or delete this file at `projects/hello-world.md`.

To add a new project, create a `.md` file in the `projects/` directory:

```yaml
---
title: Project Name
date: YYYY-MM-DD
slug: project-slug
description: A short summary shown on the list.
url: https://example.com
tags: [tag1, tag2]
year: 2026
---
```

The filename should match the `slug` field (e.g. `slug: my-project` → `projects/my-project.md`).

## Writing about a project

Use this space for anything — a detailed writeup, a retrospective, a list of collaborators, a stack breakdown.

- **url** links to an external page (shown as "Visit project →" on the entry page)
- **year** is shown in the list view instead of the full date
- **tags** make the list filterable
