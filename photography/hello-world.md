This is the first photography entry. Edit or delete this file at `photography/hello-world.md`.

To add a new entry, create a `.md` file in the `photography/` directory:

```yaml
---
title: Series or Photo Title
date: YYYY-MM-DD
slug: entry-slug
location: Mexico City, Mexico
year: 2024
url: https://example.com
tags: [street, color]
---
```

The filename should match the `slug` field (e.g. `slug: mexico-city-2024` → `photography/mexico-city-2024.md`).

## Writing about photography

Use this space for anything — the story behind the photos, the light, the place, the moment.

- **location** is shown as a subtitle in the list and entry views
- **url** links to an external page (shown as "View photos →" on the entry page)
- **year** is shown in the list view instead of the full date
- **tags** make the list filterable
